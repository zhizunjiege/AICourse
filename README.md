## 如何运行

为了运行 `tests` 下的测试脚本，先设置 `PYTHONPATH` 环境变量：

### Bash

```bash
export PYTHONPATH=./
```

### Powershell

```powershell
$env:PYTHONPATH="./"
```

### Cmd

```dos
set PYTHONPATH=./
```

该环境变量只存在于当前终端会话中，目的是支持 test 文件中的模块导入，对系统无其他副作用。
